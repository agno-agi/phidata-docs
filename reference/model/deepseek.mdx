## Example

<CodeGroup>

```python agent.py

from phi.agent import Agent
from phi.model.deepseek import DeepSeekChat

agent = Agent(
    model=DeepSeekChat(),
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("Give me in-depth analysis of NVDA and TSLA")
```
