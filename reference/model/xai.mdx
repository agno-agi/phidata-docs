## Example

<CodeGroup>

```python agent.py
from phi.agent import Agent
from phi.model.xai import xAI

agent = Agent(
    model=xAI(id="grok-beta"),
    instructions=["Use tables to display data."],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("What is the stock price of NVDA and TSLA")
```

</CodeGroup>
