## Example

<CodeGroup>

```python agent.py
from phi.agent import Agent
from phi.model.vertexai import Gemini

agent = Agent(
    model=Gemini(id="gemini-1.5-flash"),
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("What is the stock price of NVDA and TSLA")
```

</CodeGroup>
